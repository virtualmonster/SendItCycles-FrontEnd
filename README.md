# CycleShop Frontend

React-based frontend application for the CycleShop e-commerce platform..

## Technology Stack
- React 18
- Vite
- TailwindCSS
- Zustand (state management)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Docker

```bash
docker build -t cycleshop-frontend .
docker run -p 80:80 cycleshop-frontend
```
